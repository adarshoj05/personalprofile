# Responsive Personal Resume (Bootstrap 5)

This project is a responsive personal resume website built with **Bootstrap 5**.

## Files
- `index.html` - Main page (sections: Home, About, Education, Skills, Projects, Contact)
- `style.css` - Custom styling

## Run
Open `index.html` in a browser.

(Optionally, you can run a local server if your environment blocks file:// resources.)

## Notes
- Navbar collapses into a hamburger menu on smaller screens.
- The contact form is a demo (no backend connected).
